Design Section 3: Color Cleanup

Move hardcoded colors into theme tokens. Dynamic color handles chrome; extended tokens handle game content.

The Split: Dynamic vs Fixed

Material You dynamic color drives app chrome. Fantasy-specific extended tokens stay fixed.

DYNAMIC (from wallpaper)
primary / secondary / tertiary
surface / surfaceVariant / background
primaryContainer / secondaryContainer
outline / outlineVariant
error / errorContainer
Used for: top bar, bottom nav, FAB, buttons, dialogs, scaffolds, input fields, cards backgrounds, dividers
FIXED (RealmsExtendedColors)
Crit/fumble, success/warning/info
Gold accent, rarity tiers
Bubble pairs (narrator/NPC/player/aside/system)
+ NEW: NPC dialogue palette
+ NEW: scrim overlay color
Used for: message bubbles, dice results, item rarity borders, NPC name colors, status effects, lore era dividers

New Extended Color Tokens

Added to RealmsExtendedColors:
