3. Visual Overhaul

Real visual changes — not just token swaps. Make every screen feel polished and intentional.

A
Character + Journal → full-screen pagers (not bottom sheets)
Currently all panels are ModalBottomSheets that slide up. Character and Journal become full-screen tab views inside the Scaffold. The bottom nav stays visible. Feels like a real app section, not a popup.
B
StatsPage visual rework
Convert all raw Surfaces to RealmsCard. Add proper visual hierarchy — ability scores as a grid of cards, HP/AC/level as a header row, morality bar prominent, backstory as an expandable card.
C
Remaining raw Surface → RealmsCard across all panels
BackpackCell, reward chips, trait chips, faction chips, exchange cards — finish the job the token migration started.
D
ModalBottomSheet → M3 defaults
Remove all RoundedCornerShape(topStart=28, topEnd=28) overrides. M3's BottomSheetDefaults already does this. Clean up remaining off-grid padding.
E
Tab row styling with M3 ScrollableTabRow
Character and Journal sub-tabs use M3 ScrollableTabRow + Tab with proper indicator animation. Matches Material You guidelines for tab navigation within a section.
